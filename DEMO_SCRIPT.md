# SkillSwap Demo Script

## 🎬 Live Demo Script (2-3 minutes)

### Introduction (30 seconds)
"Hi everyone! Today I'm excited to show you SkillSwap, a full-stack skill exchange platform I built. Let me walk you through how it works."

---

### Scene 1: Landing Page (20 seconds)

**Action**: Navigate to home page
- Show full landing page
- Highlight hero section
- Scroll to feature cards

**Script**:
"Let's start with the landing page. SkillSwap is designed to connect people who want to learn and teach skills. The homepage showcases the core value proposition - finding the perfect skill exchange partner based on complementary skills."

---

### Scene 2: Dashboard (40 seconds)

**Action**: Click "Get Started" or navigate to Dashboard
- Show user profile section
- Point to stats cards
- Highlight skills offered/wanted
- Show recent activity

**Script**:
"Once you sign in, you land on the Dashboard. Here you can see your profile, stats like matches found and sessions completed, and your skill profile - what you offer and what you want to learn. The recent activity feed keeps you updated on your latest interactions."

---

### Scene 3: Browse Matches (45 seconds)

**Action**: Click "Browse Matches" button
- Show matches page with search
- Filter by skill (e.g., "Figma")
- Show match cards with scores
- Click on a high-score match

**Script**:
"Now let's look at the matching system. The Matches page shows you potential partners ranked by compatibility score. This score is calculated using a smart algorithm that considers exact skill matches, synonyms, and partial overlaps. You can filter by name or skill to find specific matches."

**Action**: Point to match strength bar
"Each match card shows the partner's skills, what they want to learn, and a visual match strength indicator. Higher scores mean better compatibility."

---

### Scene 4: Request Session (30 seconds)

**Action**: Click "Request session" on a match
- Show toast notification
- Show status change to "Session Booked"

**Script**:
"When you find a good match, you can request a session with one click. The system automatically creates a session entry and updates the match status. You'll get a confirmation toast right away."

---

### Scene 5: Manage Sessions (45 seconds)

**Action**: Navigate to Sessions page
- Show timeline view
- Show "Happening now" section
- Show "Scheduled" section
- Show "Completed" section

**Script**:
"The Sessions page gives you a complete timeline of your skill exchanges. Sessions are grouped by status - happening now, scheduled, and completed. Each session card shows the partner, topic, what you're teaching and learning, date, time, and duration."

**Action**: Click "Mark complete" on an upcoming session
"You can easily mark sessions as complete or cancel them if plans change. The timeline updates automatically."

---

### Scene 6: Mobile Responsive (20 seconds)

**Action**: Switch to mobile view (DevTools)
- Show home page on mobile
- Show matches on mobile
- Show sessions on mobile

**Script**:
"The entire application is fully responsive. Whether you're on desktop or mobile, you get the same great experience. The layout adapts seamlessly to different screen sizes."

---

### Scene 7: Technical Overview (30 seconds)

**Action**: Show code structure or architecture diagram
- Mention React + Vite frontend
- Mention Node.js + Express backend
- Mention matching algorithm
- Mention deployment

**Script**:
"Under the hood, SkillSwap is built with React and Vite for the frontend, and Node.js with Express for the backend. The matching engine uses a three-tier scoring system with synonym recognition. Both frontend and backend are deployed on Vercel and Render respectively."

---

### Conclusion (20 seconds)

**Action**: Return to landing page
- Show final call-to-action

**Script**:
"And that's SkillSwap! A complete skill exchange platform that makes it easy to find partners, schedule sessions, and track your learning journey. Thanks for watching!"

---

## 🎥 Recording Tips

### Before Recording
1. **Clear browser cache** - Ensure fresh data
2. **Disable extensions** - Clean browser view
3. **Set browser zoom** - 100% for consistency
4. **Check API connectivity** - Verify backend is running

### During Recording
1. **Speak clearly** - Moderate pace, articulate
2. **Use mouse smoothly** - No jerky movements
3. **Highlight key elements** - Use cursor to guide attention
4. **Wait for loading** - Don't rush API calls
5. **Keep consistent timing** - Follow script timing

### After Recording
1. **Review footage** - Check for glitches
2. **Add captions** - Improve accessibility
3. **Add background music** - Optional, keep subtle
4. **Export in HD** - 1080p recommended
